Загрузите все файлы .xls для всех внутренних диапазонов почтовых индексов, используя эту ссылку: https://www.ups.com/us/en/support/shipping-support/shipping-costs-rates/retail-rates.page
Чтобы загрузить файл .xls для определенного диапазона почтовых индексов, введите любой номер почтового индекса из этого диапазона в форму и нажмите Загрузить диаграмму.

Список всех диапазонов почтовых индексов UPS смотрите в файле Диапазоны зон перевозчиков, страница «UPS zip ranges».
Пример: Чтобы загрузить файл .xls для диапазона zip 00400-00599, введите zip 00400, или 00401, или 00402, … в форму и нажмите Download Chart.
Убедитесь, что диапазон zip в загруженном файле совпадает с диапазоном zip в списке диапазонов зон Carriers.
Список диапазонов зон имеет значение 01000-01199:

Если мы скачаем файл зоны с почтовым индексом 01000, то в загруженном файле мы увидим, что он содержит зоны для zip диапазона от 01001 до 01099:

Это означает, что нам также необходимо скачать файл зоны для 01100-01199 zip диапазона, и исправить список zip диапазонов в файле Carriers zone ranges - для разделения этого диапазона.

Преобразуйте все файлы xls в файлы xlsx, используя какой-либо инструмент преобразования, например https://cloudconvert.com/xls-to-xlsx.

Collect input files for all carriers:
UPS
Download all .xls files for all domestic zip ranges using this link: https://www.ups.com/us/en/support/shipping-support/shipping-costs-rates/retail-rates.page
To download .xls file for some specific zip range, enter any zip number from this range to a form and press Download Chart.

The list of all UPS zip ranges see in the file Carriers zone ranges, page “UPS zip ranges”.
Example: To download .xls file for 00400-00599 zip range, enter zip 00400, or 00401, or 00402, … to the form and press Download Chart.
Check that zip range in the downloaded file matches to zip range in the list in the file Carriers zone ranges.
Zone ranges list has a value 01000-01199:

If we download the zone file using 01000 zip code, then in the downloaded file we can see, that it contains zones for zip range from 01001 to 01099:

It means that we also need to download zone file for 01100-01199 zip range, and fix zip ranges list in the file Carriers zone ranges - to split this range.

Convert all xls files to xlsx files using some converting tool, for example https://cloudconvert.com/xls-to-xlsx.
